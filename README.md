# Rural-Water-Dashboard-
Safely Managed Drinking Water Access Dashboard
An interactive Streamlit dashboard that visualises global access to safely managed drinking water services from 2000 to 2022.
Dataset
People using safely managed drinking water services (% of population)
Source: World Bank - World Development Indicators
Indicator Code: SH.H2O.SMDW.ZS
Collected by: WHO/UNICEF Joint Monitoring Programme
Features

Choropleth map showing water access by country with year selection
Trend analysis by region and income group
Country comparison (up to 10 countries)
Distribution analysis with histograms and box plots
Sidebar filters for region, income group, and year range
Data search and CSV download

Live App
View the dashboard on Streamlit Cloud
Tech Stack

Python
Streamlit
Plotly
Pandas

How to Run Locally
pip install -r requirements.txt
streamlit run app.py
